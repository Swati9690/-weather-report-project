# -weather-report-project

Built a responsive weather web app using HTML, CSS, and JavaScript to fetch real-time weather  data for any city or the user’s current location.
It allows users to search for any city and view the current weather details such as temperature, humidity, wind speed, and weather conditions — all fetched in real-time using a public weather API.

Tech Stack :

* HTML5 – Structure of the web page
* CSS3 – Styling and responsive layout
* JavaScript (ES6) – Logic, API integration, and interactivity
* OpenWeatherMap API – To fetch live weather

Features :

 * Search weather by city name
 * Real-time temperature, humidity, and wind speed
 * Dynamic weather icons and background changes
 * Error handling for invalid city names
 * Responsive design for all screen sizes
